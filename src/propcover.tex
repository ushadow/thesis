\def\abstract{
We propose a systematic hierarchical approach to continuous gesture analysis
using a unifying framework based on abstract hidden Markov models (AHMMs).
With this framework, we will develop a gesture-based interactive interface that
allows users to do both manipulative and communicative gestures without
artificial restrictions, and hence enabling natural interaction.}
